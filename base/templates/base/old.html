Trading process

  • 1. Before market open
    1. Add symbols to rtd.xlsx file
    2. Create excel rtd analysis sheet
    3. Check economics calendar
    4. Read market email
    5. Market movement opinion
    6. Stock weekday opinion (symbols)
    7. Spiritual pray opinion
  • 2. Trading hours
    1. After market open
      • Close opinion (close/symbol)
      • Close profit positions
    2. During market hours
      • Monitor holding positions
      • Review market news
      • Monitor excel rtd (symbols)
      • Review live news (symbols)
      • Seek opportunity (part 4)
      • Manage position when necessary
      • Position opinion (open/symbol)
      • Close opinion (close/symbol)
    3. Before market close
      • Open positions
      • Close profit/loss positions
  • 3. After market close
    1. Export then import statement
    2. Update google/yahoo stock
    3. Position analysis (symbols)
    4. Option timesale report (symbols)
    5. Behavior opinion
  • 4a. Seek opportunity
    • Most active options
    • Hot news that affect fundamental
    • Top/Worst Movers
    • Custom Screener
    • Current watchlist symbols
    • Company events (earnings/dividend/others)
    • More ways to come...

    4b. After you have idea or symbols to trade, ready symbol then backtest

    4c. Enter open position before market close (2c.)

Ready symbol

  • 1. Provide underlying data
    1. Get thinkback csv
    2. Get earning/dividend
    3. Add underlying symbol
    4. Add split history
  • 2. Import all then clean
    1. Import earning/dividend
    2. Import stock/options
    3. Import weekdays then calc IV
    4. Import google/yahoo
    5. Set final/optionable
    6. Enable symbol
  • 3. Create opinions
    1. Open symbol manage
    2. Market valuation (monthly)
    3. Market indicators (weekly)
    4. Stock industry analysis (monthly/symbol)
    5. Stock fundamental analysis (season/symbol)
  • 4. Backtest research
    1. Idea of trading or direction
    2. Select formula then backtest (repeat)
    3. Review the report then select result
    4. Select strategy then backtest (repeat)
    5. Review the report then use the trade
  • 5. Decision making, condition met
    1. Significant backtest result
    2. Valid stock opinion
    3. Both must same direction or correlate
    4. Analysis previous positions
    5. Ready to open position
    6. Include symbol in Watchlist DB
    7. Wait for a few days
    8. Enter position

Season maintainance

  • 1. Download then import csv data (active symbol)
  • 2. Update and clean data (active symbol)
